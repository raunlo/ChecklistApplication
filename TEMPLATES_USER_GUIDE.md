# Templates Feature - User Guide

## What are Templates?

Templates are **reusable checklist blueprints**. Instead of manually creating the same checklist items every time, create a template once and use it repeatedly.

**Example:** You go grocery shopping every week with the same items. Create a "Grocery" template, then reuse it 52 times a year instead of typing the same list each time.

---

## 📖 How to Use Templates

### STEP 1️⃣: Navigate to Templates
```
Open the app → Click "Templates" in navigation
OR go to: /templates
```

**You'll see:**
- Empty state if no templates yet (with helpful example)
- Grid of template cards if you have templates
- "New Template" button to create

---

### STEP 2️⃣: Create a Template
```
Click "New Template" button
  ↓
Fill in:
  • Template Name (required): e.g., "Weekly Grocery"
  • Description (optional): e.g., "Items for weekly shopping"
  ↓
Click "Create"
```

**After creation:**
- You're taken to template detail page
- Template appears in your list
- Ready to add items

---

### STEP 3️⃣: Add Items to Template
```
On template detail page:
  1. Click "Edit" button (top right)
  2. Items section becomes editable
  3. Type in the "Add new item..." field
  4. Press Enter or click + button
  5. Item appears in the list below
  6. Repeat for each item you want
  7. Click "Save" when done
```

**In Edit Mode:**
- ✏️ See text "No items yet. Add one above!"
- 🎯 Input field shows "Add new item..."
- ❌ Red trash icon appears on hover to remove items

---

### STEP 4️⃣: View Your Template
```
After saving:
  • See all items in the template
  • See item count: "Items (5)"
  • See when items were added
  • Ready to use!
```

---

### STEP 5️⃣: Use the Template (Create Checklist)
**On the Checklists page:**
```
Click "Apply Template" or "Create from Template" button
  ↓
Select your template
  ↓
Preview items that will be added
  ↓
Confirm
  ↓
NEW CHECKLIST CREATED with all template items! ✅
```

---

## 💡 Tips & Tricks

### Organizing Templates
- Name templates clearly: "Morning Routine", "Packing Trip", "Before Vacation"
- Use descriptions: "Items needed for home cleaning"
- Group similar templates in your mind

### Creating Effective Templates
✅ **Good template items:**
- Recurring things you do repeatedly
- Standard checklists (shopping, packing, cleaning)
- Things that never change week-to-week

❌ **Avoid:**
- One-time items
- Items that change frequently
- Too many items in one template (50+)

### Using Templates Efficiently
1. Create 5-10 templates for your common tasks
2. Use them from the Checklists page
3. Edit the checklist after creating if needed
4. Save time by not retyping!

## ❓ Common Questions

### Q: Can I edit a template after creating it?
**A:** Yes! Go to template detail → Click "Edit" → Change name, description, or items → Save

### Q: What happens if I delete a template?
**A:** The template is deleted, but checklists created from it stay intact. It's safe to delete.

### Q: Can I have duplicate items in a template?
**A:** Yes, but it's not recommended. The app will allow it though.

### Q: How many items can a template have?
**A:** Unlimited, but 20-30 items is ideal. Too many templates become hard to manage.

### Q: Can I duplicate a template?
**A:** Not yet - create a new one and copy items manually. (Feature coming soon!)
